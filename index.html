Bulk CSV Import Manager

Import Billets or Personnel records locally in the browser. Format your spreadsheets with the exact headers listed in each section, export as CSV, and select the file.

1. Import Billets CSV

Populate organizational positions before assigning personnel.

Expected CSV Column Headers:
  • Code (Required Billet code, e.g. ME-MIL-01)
  • Title (e.g. Program Director)
  • Program (ME or SE)
  • Type (Military or Civilian)
  • Role (Faculty or Staff)
  • Required Rank (e.g. Maj, Professor)

2. Import Personnel CSV

Upload roster members and optionally assign them immediately.

Expected CSV Column Headers:
  • Name (Required full name)
  • Rank (e.g. Maj, GS-13)
  • Type (Military or Civilian)
  • Role (Faculty or Staff)
  • Arrival Date (Tour start date)
  • Departure Date (Tour end date)
  • Assigned Billet Code (Optional matching code)
